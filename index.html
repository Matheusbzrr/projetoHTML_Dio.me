Compartilhando o aprendizado

Sobre as tags h1 até h6 e listas ordenadas ou não.

exemplos:

  • h1: Usada para o título principal da página ou seção. Deve ser única por página e representa o conteúdo mais importante.
  • h2: Usada para subtítulos diretos abaixo de h1. Geralmente marca seções principais dentro do conteúdo.
  • h3: Marca subseções dentro de uma seção definida por um h2. É útil para detalhar tópicos secundários.
  • h4: Usada para cabeçalhos de nível inferior, subdividindo o conteúdo de uma seção h3.
  • h5: Define títulos ainda mais específicos, geralmente detalhes ou tópicos menores dentro de uma subseção h4.
  • h6: É o cabeçalho de menor importância, usado para tópicos muito específicos ou menos relevantes dentro de uma seção h5

Utilizano a tag h4 nesse cabeçalho dentro das tags h1, h2 e h3 para falar sobre as UL, Ol e LI

UL são listas não-ordenadas como a lista a cima e essa aqui a baixo
  • Item 1
  • Item 2
  • Item 3
OL são listas ordenadas como essa a baixo
  1. Item 1
  2. Item 2
  3. Item 3
LI Define um item dentro de uma lista (e por padrão o LI é uma lista não ordenada) <ol> ou <ul>.
